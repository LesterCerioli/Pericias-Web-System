UCEntidade
  - EntEndereco
     - Classe nova no UCEntidade. Possui métodos de consulta de CEP por string ou por um objeto EstCEP.  Estes métodos retornam uma EstEndereco com: CEP, Nome do logradouro*, EstPais, EstUnidadeFederacao, EstCidade, EstBairro* e EstTipoLogradouro* (* se o CEP não for único para a localidade). As estruturas internas à classe EstEndereco são preenchidas com dados do UC.
     
WebServicePadrao
  - BaseWebService
    - Foram criados dois métodos novos: RegistrarUsuario e RegistrarSistema. O método anterior (RegistraUsuario), continua existindo para manter compatibilidade. O ideal é que todos alterem para RegistrarUsuario e, assim que a segurança de WebServices estiver completamente disponível, modificar para RegistrarSistema. A diferença entre estes métodos é que o RegistrarSistema faz autenticação no Sistema de Segurança e possui um parâmetro a mais que é o código do usuário que fez login no sistema, para que apareça o nome correto no log do Oracle.
    - Implementado o método Autorizado, que recebe como parâmetro o nome do WebMethod a ser executado e retorna 4 possíveis resultados do Enum RespostaAutorizacao: Autorizado, NaoAutorizado, NaoLogado e SessaoDesabilitada. Esse é o método que deve ser executado para verificar se um determinado sistema pode ou não acessar um método do WebService.
    - Será necessário incluir no WebService uma key no appSettings chamada "SegWebService", contendo a url do WebService de Segurança que o sistema irá acessar.
